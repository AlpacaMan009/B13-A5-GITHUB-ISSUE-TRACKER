1️⃣ **What is the difference between var, let, and const?**
**Answer:** 
 "var" is function scoped and it can be changed or redeclared after declaring once.

 "let" is Block-scoped and can be updated later.

 "const" is also block scoped but cannot be updated. If a variable is declared using const then another variable of a same name cannot be declared using const.


2️⃣ **What is the spread operator (...)?**
**Answer:** Spread operator is used to expand an array or object into individual elements. It is mostly used to copy and merge arrays or objects and also used for converting array-like objects into real array.


3️⃣ **What is the difference between map(), filter(), and forEach()?**
**Answer:**
map() - It returns a new array of the same length after transforming each element.

filter() - Removes unwanted items from the array and returns only filtered items.

forEach() - Usually used to iterate through array-like objects (DOM updates, API calls)


4️⃣**What is an arrow function?**
**Answer:** This is a short syntax for writing functions using "=>". This is usually used for callbacks, event handlers.


5️⃣ **What are template literals?**
**Answer:** This is a way of expressing strings using backticks(``) instead of quotation marks.
